**The Pressure** — это профессиональный инструмент для контроля исполнения задач, управления ожиданиями и трекинга сроков. Приложение идеально подходит для менеджеров и тимлидов, которым нужно контролировать работу других сотрудников, собирать аналитику по задержкам и быстро напоминать коллегам о зависших задачах с помощью встроенной нейросети.

## ✨ Ключевые возможности

- 📋 **Управление проектами и задачами**: Создавайте проекты, разбивайте их на задачи и следите за статусами.
- ⏳ **Трекинг ожиданий (Waits)**: Делегировали задачу? Запустите таймер ожидания ответа от сотрудника.
- 🚨 **Динамические зоны риска (Зеленая/Желтая/Красная)**: Визуальный контроль просрочек. Если сотрудник долго не отвечает, таймер краснеет.
- 🤖 **AI-ассистент (LLM)**: Автоматическая генерация вежливых (или настойчивых) напоминаний сотрудникам с учетом контекста и количества просрочек через **OpenRouter**.
- 🏆 **Стена позора**: Сводная аналитика по всем активным задержкам в команде с сортировкой по самым злостным нарушителям.
- 👤 **Мультиаккаунтность**: Поддержка нескольких пользователей на одном устройстве. Каждый пользователь работает в своей изолированной SQLite базе данных.

## 🚀 Установка и запуск (Windows)

Для работы приложения на вашем компьютере должен быть установлен **Python 3.10+**.

1. Склонируйте или скачайте последний [релиз](https://github.com/Duneholy/pressure/releases) репозитория.
2. Откройте папку с проектом.
3. **Важно:** Обязательно распакуйте скачанный `.rar` архив в удобную папку перед запуском.
3. Запустите файл `run_windows.exe` двойным щелчком.

Скрипт полностью **автоматизирован**:
- Проверит наличие Python.
- Создаст виртуальное окружение (`.venv`).
- Установит все необходимые зависимости из `requirements.txt`.
- Запустит локальный сервер приложения.

После успешного запуска приложение откроется в браузере.

## ⚙️ Настройка AI (OpenRouter)

Чтобы приложение могло само генерировать тексты напоминаний, потребуется API-ключ от любого LLM-провайдера (через OpenRouter).

1. Зарегистрируйтесь на [OpenRouter](https://openrouter.ai/) и получите API ключ.
2. В приложении зайдите в **Настройки** (шестеренка в правом верхнем углу).
3. Введите свой ключ и выберите модель (например, `minimax/minimax-m3`). Можно выбрать бесплатную.
4. Нажмите **Сохранить**. Эти данные будут сохранены локально и безопасно только в вашей личной базе данных!

## 🔒 Безопасность и архитектура

- **Local-first**: Все ваши данные (проекты, задачи, сотрудники, настройки) хранятся исключительно локально в папке `data/users/`. Они никуда не отправляются (за исключением текстов задач при генерации сообщений через LLM).
- **Изоляция**: Для каждого пользователя при регистрации создается собственная БД SQLite.
- **FastAPI Backend**: Быстрый, современный и асинхронный сервер.
